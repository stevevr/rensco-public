Streetcart

Streetcart - a world where the high street is thriving and local search for products is as accessible and effective as online supermarkets like amazon and more rewarding.
Real-time Application built with Meteor, MongoDB for Web, iOS and Android. Rensco also custom built a scalable Web Crawler using Node.js & Phantom JS on a Docker containerised infrastructure.
